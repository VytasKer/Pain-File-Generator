# Pain XML Generator

This is a **Streamlit application** for generating large `pain.001.001.03` and `pain.001.001.09` XML files. It allows users to define payment instances and customize header details, debtor information, and advanced inputs for testing purposes.

---

## Application Features
- **Streamlit-based user interface** for ease of use.
- Supports customization of payment information and debtor details.
- Enables large-scale XML file generation.
- Includes download functionality for generated XML files.
